# togroup-ui

## Install

```
npm install togroup-ui
```

## Usage

To use togroup-ui,in your main.js

```javascript
import Vue from "vue";
import TogroupUI from "togroup-ui";
import "togroup-ui/lib/togroupui.css";
Vue.use(TogroupUI);
```
